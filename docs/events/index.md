# 活动与赛事（Xbotics 社区）

> [!NOTE]
> 本章节用于记录和沉淀基于 Xbotics-Robot-Claw 开展的各类活动，包括：Workshop、课程实践、比赛、线上分享等。

## 1. 如何使用本章节

建议对每类活动建立一个小节，并包含以下信息：

- 活动名称与时间；
- 组织方与主要面向人群；
- 主要内容简介（例如：教学大纲、实践任务）；
- 相关资料链接（讲义 / 录像 / 作业 / 代码仓库等）；
- 复用建议（如果其他老师 / 组织者想复用这个活动，应该怎么改造）。

## 3. 如何发起新的活动

如果你希望基于本项目在 Xbotics 社区内发起新的活动，建议：

1. 先在 GitHub Issue / 社区论坛中发起讨论，简要说明：
   - 目标人群；
   - 时间与形式（线上 / 线下）；
   - 预期产出（例如：一套作业、一个 demo 视频等）。
2. 活动结束后，将关键资料整理到本章节对应的小节中；
3. 如有录屏 / 公共分享材料，也可以在 README 或首页中添加醒目链接。
