## 1. 收录标准与使用方式

- **收录标准**（可随社区共识调整）：
  - 与 OpenClaw / Picoclaw 有清晰的结合方式（代码、技能或文档中有说明）；
  - 有公开仓库或文档可查，便于他人复现或借鉴；
  - 在平台、任务类型、教学价值等方面具有一定代表性。
- **你可以这样使用本列表**：
  - 浏览不同条目，按平台 / 难度 / 应用场景筛选适合自己的项目；
  - 作为课程 / 活动设计时的项目池；
  - 如果你已完成不错的项目，也可以按本页格式提交 PR 收录。

后续可以考虑增加标签（如 `#mobile-base`、`#arm`、`#claw`、`#course-project`、`#competition` 等）来做进一步分类。

## 2. 项目清单

### 2.1 HomeBot × OpenClaw：家庭服务机器人实验平台

- **项目链接**：  
  - 仓库：<https://github.com/choco-robot/HomeBot>
- **一句话简介**：  
  - 基于宜家小推车 + 全向轮 Kiwi 底盘 +（可选）机械臂，构建一台可移动、可抓取的家庭服务机器人平台，通过 HomeBot 统一管理底盘 / 机械臂 / 视觉，再与 OpenClaw / Picoclaw 结合，实现「大模型控制的家庭机器人」。
- **类型与场景**：
  - 类型：开源机器人平台 + 教学 / 研究 demo；
  - 场景：家庭 / 室内环境中的移动、跟随、抓取等任务。
- **与 OpenClaw 的结合方式（概述）**：
  - 把 HomeBot 的机器人控制能力封装为 OpenClaw / Picoclaw 中的一个「技能」；
  - 通过自然语言指令（例如「让机器人前进 20 厘米」「右转 90 度」「机器人看到了什么？」）触发底盘运动、机械臂控制和视觉查询。
- **适合人群与上手建议**：
  - 已有一定机器人 / 嵌入式硬件基础，希望搭建具身智能实验平台的开发者；
  - 高校课程大作业 / 研究 demo 的硬件平台选择之一；
  - 建议先在本仓库的「项目概览」章节理解整体架构，再详细阅读 HomeBot 仓库文档。
- **相关活动 / 教程**：
  - 如有基于 HomeBot 的 Workshop、课程或比赛，可以在「活动与课程 / 赛事」章节中补充对应条目，并在此处交叉链接。
